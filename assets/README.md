# Assets

Drop live dashboard screenshots here so they render in the main README:

- `dashboard_light.png` — the console in light mode (Live Console tab)
- `dashboard_dark.png` — the console in dark mode
- `live_console.png` — a processed message showing all four result cards

**How to capture:** run `streamlit run app.py`, open the app, and use your OS
screenshot tool (macOS: Cmd+Shift+4). Then uncomment the image lines in `README.md`.
